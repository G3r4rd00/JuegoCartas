# Casas en Guerra (MVP)

Prototipo web 1 vs 1 de cartas de tropas. La unica condicion de victoria es destruir la fortaleza rival.

## Concepto del MVP

- Cada jugador empieza con 20 de fortaleza.
- En cada turno recibes 5 de oro.
- Juegas tropas desde la mano pagando su coste.
- Las tropas atacan tropas enemigas o la fortaleza.
- Si el rival tiene unidades con Guardia, debes atacarlas primero.
- Gana quien deja la fortaleza rival en 0.

## Modos

- Humano vs Humano (mismo dispositivo)
- Humano vs Maquina

## Ejecutar

```bash
npm start
```

Abrir: http://localhost:3000
